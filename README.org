** Installation

   Alps is available through [[http://cocoapods.org][CocoaPods]] it, simply add the following
   line to your Podfile:

#+BEGIN_SRC ruby
  pod 'AlpsSDK', :git => 'https://github.com/MatchMore/alps-ios-sdk.git', :tag => '0.4.0'
#+END_SRC
